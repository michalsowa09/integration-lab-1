# Projekt Laboratorium 1 - Zadanie "Django Setup"
Celem tego laboratorium jest **poprawne przygotowanie środowiska programistycznego** oraz konfiguracja systemu kontroli wersji Git.

### Wymagania techniczne:
* Zainstalowany Python 3.x
* Skonfigurowane klucze SSH
* System Git

### Jak uruchomić projekt:
Aby uruchomić serwer, używam komendy:
`python manage.py runserver`

Więcej informacji o technologii znajdziesz tutaj: [Oficjalna dokumentacja Django](https://docs.djangoproject.com/en/5.0/)
